  • UTDesign Tool Lookup

    This tool was developed to assist students and employees in locating
    shop tools in my place of employment, the UTDesign fabrication shop.
    This tool makes use of jQuery and JSON in order to store, retrieve, and
    display tool location data. The JSON can be easily edited to allow for
    expansion of our shop's tool inventory and continued assistance to those
    who just need to find that one obscure tool!

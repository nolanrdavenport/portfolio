  • Tiny Tanks with Networking

    This is not only a simple tiny tanks game, but one that supports networking!
    I developed both a server program that hosts the connection between the players
    and a client program that allows the user to control their vehicle. This has only
    been tested between different computers in a private network, but I am sure with some
    port forwarding, you can actually connect people in different households.
